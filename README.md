**FrontEnd Setup**

To run FrontEnd, you need to have Node.js installed on your machine. If you don't have it, you can download it [here](https://nodejs.org/en/).

After that follow the steps below:

```bash
cd frontend
cd time2gather
npm install
npm run dev
```
